Security Best Practice: Use Stripe Restricted API Keys (RAK)

For maximum security, do not provide your account's master Secret Key. Instead, go to your Stripe Dashboard and create a Restricted API Key (RAK) with the following permissions:

  • Checkout Sessions: Write
  • Payment Intents: Read
  • Webhooks: Write
